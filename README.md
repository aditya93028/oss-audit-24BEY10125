## Project Description
This project focuses on analyzing Python as an open source software. It includes study of its origin, philosophy, license, ecosystem, and Linux implementation. The project also demonstrates basic shell scripting skills.

## Scripts Included

1. System Identity Report  
   Displays system details such as user, kernel version, date, and uptime.

2. Package Inspector  
   Checks whether a package is installed and displays its version.

3. Disk and Permission Auditor  
   Shows directory size and permissions.

4. Log Analyzer  
   Counts number of error lines in a log file.

5. Manifesto Generator  
   Takes user input and generates a text file.
